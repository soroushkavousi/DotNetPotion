# AppEnvironment

`AppEnvironment` effortlessly identifies the current environment in which your application is running. (e.g., `AppEnvironment.IsProduction`).

## Features

- Access to **Properties**
    - `EnvironmentName`
    - `IsDevelopment`
    - `IsTesting`
    - `IsStaging`
    - `IsProduction`
- Access to **Method**
  - `bool IsEnvironment(string environmentName)`
